Information

Desktopify has two separate interfaces, the app management interface and the web app or browser interface.

The app management interface allows you to desktopify new web apps, create shortcuts for new or existing web apps, and run apps. You can also change settings such as the app icon, color, name and what url's the app contains.

As an example, you might create an app named "Development" containing a tab for your Trello board and another tab for your Github repo. Now you have easy access to the websites you need for development without cluttering up your browser.

Use Cases

  • Create a dedicated Gmail app and pin it to your taskbar for easy access.

  • Create a dedicated app for your wordpress dashboard to make editing content and posts easy.

  • Put all the tabs you need for work - time management, email client and more - into one app.

  • Constantly searching? Create a dedicated search app to host all those search results.

  • Students can pin a Canvas app to their start menu and have easy access to all class information.

Development

Desktopify is currently in development and does not have a stable release. You can follow development on Github here.

Note that the UI is in an early state and many cosmetic and usability improvements are on the way.
